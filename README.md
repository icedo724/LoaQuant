# 로스트아크 시장 동향 분석 (LoaQuant)

## 주요 거래품목 시세 분석
GitHub Actions를 활용해 로스트아크 경매장의 주요 품목 최저가를 **1시간 주기**로 자동 수집하고 시각화합니다.

### 가격 수집 대상
1. **T3/T4 강화 재료**
2. **생활 재료**
3. **배틀 아이템**
4. **유물 각인서**
5. **T4 8레벨 이상 보석**

### 서비스 링크
수집된 데이터는 Streamlit을 통해 대시보드에서 실시간으로 확인할 수 있습니다.
**[LoaQuant 바로가기](https://loaquant.streamlit.app/)**

### 주요 기능
**1. 시세 추적**
- 1시간 단위 실시간 최저가 흐름 차트 제공
- **로아 서버 기준(매일 오전 06시)** 으로 정산된 일평균 가격 제공
- 전일 대비 등락폭 및 가격 변동 표시

**2.  시장 분석**
- **RSI (상대강도지수)** 및 **볼린저 밴드**를 활용한 통계적 분석
- **과매수(고점)/과매도(저점)** 구간 탐지
- 직관적인 시장 분석 리포트 카드 UI 제공

**3. 시각화 및 유틸리티**
- **정기 점검 시각화**: 매주 수요일 점검 시간(06:00~10:00) 차트 내 회색 영역 표시
- **이벤트 추적**: 주요 이벤트 종료 일자 그래프 내 마킹
- **교환 효율 계산**: 하위 재료 ↔ 상위 재료 교환 시 손익(골드 가치) 자동 비교 분석
